**What is this migration dataset?:** This dataset contains over 1000 2D timelapse microscopy movies of migrating neutrophil granulocytes, as part of a comprehensive compound screening project to demonstrate our high throughput array microscope, ComplexEye. See our [Nature Communication paper](https://www.nature.com/articles/s41467-023-43765-3) for details.


[Explore the dataset with interactive segmentation and tracking results](https://cellmigration.isas.de/dataset/1)
